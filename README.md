# RepoInsight AI 🚀  
### RAG-based Code Intelligence System

> ⚠️ Status: Planning Phase

🔗 https://github.com/Kalash-Harchandani/repoinsight-ai

RepoInsight AI is a planned system to analyze GitHub repositories using Retrieval-Augmented Generation (RAG), enabling semantic search and AI-powered code understanding.

## ⚙️ Tech Stack
Node.js, Express, React, Pinecone, Gemini API

## 🚀 Planned Features
- RAG pipeline for repository-level semantic querying  
- Pinecone-based vector search with top-k retrieval  
- Gemini-powered explanations of code logic and architecture  

## 🎯 Goal
Make large codebases easy to understand through AI-assisted exploration.
